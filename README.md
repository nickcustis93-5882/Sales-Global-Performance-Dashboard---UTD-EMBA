# 📊 Sales Global Performance Dashboard
**Executive MBA - Managing IT in the Analytics Age**

This GenAI-powered dashboard provides high-level visibility into global sales performance metrics, integrating automated data visualization with a strategic operations framework.

## 🚀 [Launch Live Dashboard](https://nickcustis93-5882.github.io/Sales-Global-Performance-Dashboard---UTD-EMBA/)

---

### 🎯 Objective
To streamline decision-making for executive leadership by providing real-time, interactive insights into regional sales trends and performance KPIs.

### 🛠️ Key Features
* **Dynamic Visualizations:** Interactive charts powered by Google Charts.
* **Global Scope:** Consolidated data views across multiple international markets.
* **Strategic Alignment:** Designed to facilitate the "OODA Loop" (Observe, Orient, Decide, Act) for faster operational response.
